Boa pergunta, Rafael! 🚀
No HTML/CSS não temos unidade mm real (milímetros físicos), porque a renderização depende da densidade de pixels do monitor. Ou seja: 175mm só será exato em impressão, não em tela.

👉 Para usar no site sem distorcer você deve trabalhar em proporção fixa (a imagem é quadrada).
Assim você garante que sempre ficará proporcional:

Exemplo 1 – HTML direto

html
<img src="logo_175x175mm.png" alt="Logo Taís Moreira" width="175" height="175">

  • Aqui width e height são em pixels, não mm.

  • Mas como a imagem foi exportada em 300DPI, quando imprimir ela terá exatamente 175x175mm.

Exemplo 2 – CSS mantendo proporção

html
<img src="logo_175x175mm.png" alt="Logo Taís Moreira" class="logo">

css
.logo {
  width: 175px;   /* largura exibida */
  height: auto;   /* mantém proporção sem distorção */
}

Exemplo 3 – Para impressão em mm reais

No CSS você pode usar mm se o objetivo for print (PDF ou papel):

css
@media print {
  .logo {
    width: 175mm;
    height: 175mm;
  }
}

➡️ Na tela pode não bater 1:1, mas quando imprimir vai sair exatamente em 175x175mm.

  • Para web: use width="175" e `height="1
